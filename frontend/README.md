### Run
yarn install - 의존성 모듈 설치 (node_modules 최신화)
yarn dev - dev 모드 빌드 후 개발 서버 실행
yarn build - production 모드 빌드(빌드 결과물 -> .next 디렉터리)
yarn start - production 모드 빌드 후 개발 서버 실행
